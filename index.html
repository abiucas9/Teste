A Tartaruga do Python na Web! 🐢

Desenhando uma espiral colorida sem usar um pingo de JavaScript.

import turtle # Criando a nossa tartaruga t = turtle.Turtle() # Deixando ela super rápida t.speed(0) # Desenhando uma espiral maluca cores = ["red", "blue", "green", "orange", "purple", "yellow"] for x in range(100): t.pencolor(cores[x % 6]) # Alterna as cores da lista t.width(x / 10 + 1) # Vai engrossando a linha aos poucos t.forward(x) # Anda pra frente t.left(59) # Gira quase 60 graus
